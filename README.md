# ETANIMART
Etanimart adalah web agritech yang menghubungkan petani (Petugas) dan pembeli (Pengaju). Fitur kerennya, AI Plant Health Assistant, bisa scan foto tanaman sakit untuk diagnosis dan langsung merekomendasikan obatnya. Didukung dashboard manajemen dan sistem 3 aktor yang rapi, web ini sangat potensial menghasilkan uang dari komisi transaksi.
